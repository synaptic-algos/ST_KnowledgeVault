# Navigation & Vault Explorer

> Source Features: ProductDevelopmentSupport EPIC-002 (Product Navigator) & EPIC-003 (Status Dashboard)

## Tree View
- Multi-vault selector on the left pane
- Expand nodes: Epic → Feature → Story → Task
- Context panel shows owner, status, linked tests

## Search & Filters
- Global search (⌘/Ctrl + K) for IDs, titles, owners
- Filters: status, priority, sprint, tag
