## How to use this template
- Copy this file into your epic or product folder as `PRD.md`.
- Update the front-matter fields (id = PRD identifier, seq = running number) and increment `version` with each approved change.
- Maintain the changelog list using the locked format `YYYY-MM-DD – owner – summary – affected_requirements`.
